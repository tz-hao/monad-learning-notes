# 研究卡：为什么 Morpho Blue 的借贷设计和 Aave V3 不一样？

> 建立日期：2026-07-16  
> 目标：先定义要验证的问题和证据范围，不把这张研究卡当作最终结论。

## 研究主题

**为什么 Morpho Blue 选择“许可less、孤立、参数不可变”的借贷市场，而 Aave V3 选择由治理配置的共享流动性池？这会怎样改变资金效率、风险承担者和产品体验？**

## 研究对象

- 主要对象：**Morpho Blue**，一个 EVM 上的非托管、浮动利率借贷底层协议。
- 对比对象：**Aave V3**，一个以资金池和按资产储备参数为核心的非托管借贷协议。
- 生态现象：借贷协议从“协议统一上币与管理风险”转向“底层开放建市场、上层策展风险”的模块化设计。

## 所属分类

**Lending（超额抵押借贷）**。

## 我想回答的问题

1. Morpho Blue 的一个市场由哪些参数定义？这些参数为何要在创建后不可变？
2. 与 Aave V3 的共享池相比，风险究竟从哪里隔离、又转移给了谁？
3. 无需逐个市场治理批准，为什么仍不等于“完全没有治理”？
4. 对存款人、借款人、风险策展人（curator）和前端产品而言，两个设计分别带来什么收益与成本？
5. TVL 与活跃借款数据能否支持“开放市场设计已获得真实需求”这一判断？它们又不能说明什么？

## 为什么这个问题重要

借贷不是只看利率的产品：用户把资产存入后，实际承担的是抵押品、预言机、清算和流动性风险。Aave 的路径是用治理审核资产与参数，并让同一池子的资产形成共同流动性；Morpho Blue 的路径是把单个市场拆成“一种抵押品 + 一种借款资产 + LLTV + 预言机 + 利率模型”的固定组合。

这对应真实的产品取舍：

- 借款人可能需要更高的抵押效率或更快接入的新资产；
- 存款人可能更在意自己到底暴露于哪一种抵押品与预言机；
- 策展人可以把市场组合成面向普通用户的金库，但也因此承担筛选和风险说明责任；
- 开发者可选择接入一个治理配置的综合市场，或自己搭建 / 策展细分市场。

截至 2026-07-16，DefiLlama 的 Morpho 页面显示约 **67.4 亿美元 TVL**、**35.86 亿美元活跃借款**。这说明它不是纯概念设计，但这两个动态指标不能直接证明某一种架构“更安全”或“更好”。

## 资料边界

本次会看：

- Morpho Blue 的市场参数、LLTV、预言机、利率模型、清算与市场创建规则；
- MetaMorpho / curator 在“开放底层”之上承担的风险管理角色；
- Aave V3 的资金池、超额抵押、健康因子、利率和风险隔离功能；
- DefiLlama 的 TVL、活跃借款及其口径局限；
- 官方白皮书、合约仓库和论坛中可验证的设计理由。

本次暂时不研究：

- 逐个市场、逐条链的 APY 排名与短期激励；
- MORPHO / AAVE 代币价格、估值和投资建议；
- 对所有 curator 金库做安全审计，或对具体预言机实现做代码审计；
- 清算机器人 MEV 策略、历史坏账的完整量化回测；
- Morpho 的其他产品与 Aave V4 的完整功能比较。

这样做是为了先回答“架构如何分配风险”，避免被瞬时利率或 TVL 变化带偏。

## 初始假设

我目前的猜测是：Morpho Blue 并非简单追求“比 Aave 更高利率”，而是把风险配置权从协议级资产列表拆到市场级参数，再把普通用户不愿承担的筛选工作交给 curator / vault。这样能更快创建适合特定抵押品的市场、让风险边界更清楚，也可能提高资本利用率；代价是市场更分散，用户必须判断预言机、参数和 curator 是否可信。

相对地，Aave V3 的统一市场和治理式参数管理降低了普通用户选择单个风险市场的负担，流动性体验也更集中；但新资产接入和特殊参数设计会更依赖治理与风险流程。

这是待验证假设，不是结论。

## 不确定点

1. DefiLlama 的 `Morpho V1` 页面标签与当前 Morpho Blue / 生态产品的统计范围是否完全一致？后续需核对数据归因，不能直接把全部 TVL 当成单一合约的规模。
2. “风险隔离”能隔离到什么程度？同一 curator 金库跨市场配置、共用预言机或抵押资产时，是否会形成新的相关性风险？
3. 市场参数不可变后，错误的预言机或利率模型如何被用户安全退出？流动性不足时退出成本如何衡量？
4. MetaMorpho 的 curator 实际上使用哪些选择标准、权限和应急机制？普通存款人能否清楚看见这些规则？
5. 哪些指标最适合比较两种架构的真实效率：利差、利用率、清算损失、可用流动性，还是风险调整后的存款收益？
